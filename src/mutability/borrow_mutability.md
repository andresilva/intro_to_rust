# Borrow Mutability

Borrowed pointers may coerce:

```rust
let a = &mut 1;
let b: &usize = a; // ok
let c: &mut usize = b; // error!
```

Values can be *frozen* by borrowing

```rust
let mut a = Vec::new();
{
    let b = &a; // freezes `a`
    a.push(1);  // error!
    
    println!("b: {:?}", b);
}
a.push(2); // ok
```
